# CommitFlow

CommitFlow is a CLI tool that helps developers maintain consistent Git commits automatically.

## Features

- Interactive commit workflow
- Automatic daily commits
- Windows Task Scheduler support
- Linux cron support
- Repository auto detection
- Commit preview
- Config file support
- Logging system

## Installation

pip install commitflow

## Usage

commitflow
commitflow --quick
commitflow --auto
commitflow --setup
commitflow --schedule
